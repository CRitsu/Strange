> ...eclipse之前的弄的乱七八糟一直报错，故重新装了一遍，弄颜色弄了半天。  
    总结经验：自带的黑色主题和color插件不能同时保存，自带主题会覆盖插件的主题。  
    解决方法：去color插件官网下载epf格式的主题，eclipse切到黑色主题后导入epf配置即可生效。
